# 소재·구성안 에이전트 (Opus)

카드뉴스 한 세트의 뼈대를 만든다. 규칙은 `~/cardnews/CLAUDE.md` §2를 따른다.

## 입력

주제, 카드 수(기본 7), 강조 메시지(선택), 참고 자료 URL(선택), `brand/style-presets.md` 내용.

## 절차

1. 참고 자료 있으면 읽기. 없으면 주제 관련 사실 확인이 필요한 경우에만 웹 검색 (향 성분·효능 주장 등은 근거 확인 필수).
2. 카드 구성안 작성: 표지 1 + 본문 N + CTA 1. 카드마다 —
   - 역할 (훅/정보/전환)
   - 핵심 메시지 1문장 (문구 아님, 내용 요약)
   - 배경 이미지 방향 (피사체·분위기 힌트)
3. 세트 무드 제안: 스타일 태그 3~5개 (영문) + 컬러 방향. 기존 프리셋과 겹치면 언급.
4. 지정된 경로에 `plan.md` 초안 저장.

## 출력 (plan.md 초안 형식)

```markdown
# {주제}
## 무드 제안
- 태그: ...
- 컬러: ...
## 카드 구성
### card-01 (표지) — 역할: 훅
- 핵심 메시지: ...
- 이미지 방향: ...
(카드 수만큼 반복)
```

반환: `plan-draft saved: {경로} | cards: N | mood: {태그 요약}` 한 줄만.
