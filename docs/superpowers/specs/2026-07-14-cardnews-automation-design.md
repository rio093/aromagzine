# 카드뉴스 자동화 시스템 설계

- 날짜: 2026-07-14
- 상태: 승인됨
- 구조: 하이브리드 (스킬 오케스트레이션 + 이미지 생성만 병렬 서브에이전트)

## 목적

Higgsfield MCP(이미지 생성) + Figma MCP(레이아웃 조립)를 이용해
감각적인 사진 배경의 인스타그램 카드뉴스를 반자동으로 제작한다.
목표: 세트당 4시간 이내.

## 확정 기준

| 기준 | 결정 |
|---|---|
| 용도 | 개인 채널, 향/아로마·라이프스타일 콘텐츠 |
| 비주얼 무드 | 고정하지 않음 — 세트마다 카피 단계에서 무드 제안 → 사용자 승인 |
| 규격 | 4:5 (1080×1350) 기본, 1:1 (1080×1080) 파생 지원 |
| 문체 | 카드 위치별 혼합 — 표지 명사형 종결, 본문 해요체 |
| 폰트 | Pretendard |
| 카드 수 | 기본 7 (6~8 선택 가능) |
| 저장 | 로컬 폴더 필수 + 클라우드 업로드(Drive MCP 연결 후 활성화) |

## 파일 구조

```
~/cardnews/                        ← 프로젝트 루트
├── CLAUDE.md                      ← 품질 게이트(§3)·포맷 규칙·문체 규칙·트리거 프롬프트
├── brand/style-presets.md         ← 세트별 무드 프리셋 축적
└── output/{YYYY-MM-DD}-{주제}/
    ├── plan.md                    ← 카드 구성안·카피·무드 제안 (승인 대상)
    ├── bg/card-01.png ~ card-NN.png   ← Higgsfield 배경 원본
    └── final/card-01.png ~ card-NN.png, caption.txt

~/.claude/skills/cardnews/SKILL.md     ← /cardnews 오케스트레이터 스킬
~/.claude/agents/bg-image-generator.md ← 배경 이미지 병렬 생성 서브에이전트
```

## 파이프라인

1. **카피 단계 (메인 스레드)**
   주제 입력 → 카드 구성안(표지/본문/CTA) + 카피 + 이번 세트 무드 제안을
   `plan.md`로 작성 → 사용자 승인 대기. 완전 자동 모드에서는 승인 생략.
2. **이미지 단계 (병렬 서브에이전트)**
   승인된 무드 태그 기반으로 카드별 이미지 프롬프트 생성.
   카드 수만큼 bg-image-generator 서브에이전트를 동시 실행,
   각자 Higgsfield `generate_image` 호출. 4:5 규격 고정.
3. **조립 단계 (메인 스레드)**
   Figma MCP `use_figma`로 1080×1350 프레임 생성, 배경 업로드,
   텍스트 레이어 배치 (Pretendard, 문체 규칙 적용).
4. **품질 게이트 (§3)**
   아래 5개 항목 검수. 자동 모드에서 실패 카드는 최대 2회 재작업.
5. **전달**
   PNG 내보내기 → `output/.../final/` 저장 → Figma 링크 + PNG 전체 +
   인스타 캡션 전달. 클라우드 업로드는 Drive MCP 연결 후 추가.

## 품질 게이트 기준 (§3)

| 항목 | 기준 |
|---|---|
| 가독성 | 텍스트-배경 대비 충분. 부족하면 어두운 그라데이션 오버레이 추가 |
| 안전 영역 | 텍스트가 가장자리 60px 이내로 침범 금지, 이미지 주요 피사체 위에 걸림 금지 |
| 규격 | 1080×1350 정확, 텍스트 잘림 없음 |
| 카피 | 오탈자 없음, 문체 규칙(표지 명사형·본문 해요체) 준수 |
| 이미지 결함 | AI 아티팩트(왜곡된 손, 글자 뭉개짐 등) 스크린샷 검수로 확인 |

## 트리거

기본: `/cardnews 주제: {주제}` — 옵션으로 카드 수, 강조 메시지, 참고 자료, `자동` 플래그.
`자동` 플래그 시 승인 생략 + 품질 게이트 실패 최대 2회 자동 재작업.

수정 지시 (제작 완료 후, 스킬이 처리):
- 특정 카드 재생성: "card-03 배경만 더 어두운 무드로 재생성"
- 문체 일괄 변경, 1:1 파생 내보내기, 다음 세트 스타일 지정

## 오류 처리

- Figma MCP 미인증 → 3단계 진입 전 감지, 인증 안내 후 중단 (1·2단계 산출물은 보존)
- Higgsfield 생성 실패 → 해당 카드만 재시도, 2회 실패 시 사용자에게 보고
- 자동 모드 재작업 2회 초과 → 실패 카드 목록과 함께 부분 결과 전달

## 범위 밖 (YAGNI)

- 정기 스케줄 실행 (추후 스케줄러 등록으로 확장 가능하나 v1 제외)
- 클라우드 업로드 (Drive MCP 연결 전까지 비활성)
- 마음약방/회사 브랜드 프리셋 (개인 채널 전용으로 시작)
